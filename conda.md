# Conda Snippets

## Create environment

``` conda env create -n docling-env -f ./env.yml ```

## Update environment

``` conda env update -n docling-env -f ./env.yml ```

## Remove environment

``` conda env remove --n docling-env ```

## Activate environment

``` conda activate docling-env ```

## Deactivate environment

``` conda deactivate ```

# Run Project

``` python src/main.py ```
